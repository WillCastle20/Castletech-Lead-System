# Lead Capture Dashboard

Starter structure for a lead capture dashboard using a Node.js + Express backend, SQLite database, and plain HTML/CSS/JavaScript frontend.

## Project Structure

```text
backend/
  package.json
  server.js
database/
  schema.sql
frontend/
  index.html
  styles.css
  app.js
```

## Current Scope

- Express server with basic health and placeholder lead routes.
- SQLite schema starter for future lead storage.
- Static frontend starter for an eventual admin dashboard.

Lead capture, database connection logic, scoring, and full dashboard functionality are intentionally left for later steps.

## Run Later

```bash
cd backend
npm install
npm run dev
```

Then open `frontend/index.html` in a browser.
